# Grimoire Editor - Formatting Guide

## How to Format Text

### Method 1: Using the Floating Toolbar
1. Select any text in the editor
2. A toolbar will appear above your selection
3. Click any formatting button:
   - **B** - Make text bold
   - **I** - Make text italic
   - **X** - Redact text (black bar)
   - **Pen** - Highlight text (bloody effect)

### Method 2: Using Markdown Syntax
Type these patterns directly in your text:

#### Bold Text
```
**This text will be bold**
```
Result: **This text will be bold**

#### Italic Text
```
*This text will be italic*
```
Result: *This text will be italic*

#### Redacted Text (Black Bar)
```
~~This text will be redacted~~
```
Result: ~~This text will be redacted~~ (appears as solid black bar)

#### Highlighted Text (Bloody)
```
==This text will be highlighted==
```
Result: ==This text will be highlighted== (appears with red glow)

## Keyboard Shortcuts

- **Ctrl+B** - Bold
- **Ctrl+I** - Italic
- **Ctrl+Shift+X** - Redaction
- **Ctrl+H** - Highlight

## Visual Effects

### Redaction
Creates a solid black bar over text, perfect for:
- Censored content
- Secrets to be revealed
- Mysterious elements
- Classified information

### Bloody Highlight
Creates a red glowing effect, perfect for:
- Important warnings
- Horror elements
- Blood-related content
- Emphasis with gothic flair

## Tips

1. **Combine Formats**: You can nest formats
   ```
   **==Bold and highlighted==**
   ```

2. **Quick Selection**: Double-click a word to select it quickly

3. **Stats**: Watch your word count and reading time in the bottom-right corner

4. **Professional Font**: The editor uses Crimson Text, a beautiful serif font designed for long reading sessions
