# Product Overview

**DeadLine** is a gothic horror-themed writing application that gamifies the writing process through psychological horror mechanics and typing challenges.

## Core Concept

The app transforms writing into an immersive horror experience where users face "The Deadline" - a supernatural force that drives them to write. It combines productivity tools with horror game mechanics to create urgency and engagement.

## Key Features

- **Multiple Horror Writing Modes**: Different themed editors with unique mechanics (Noctuary, Silent Vigil, Veil Typer, Verbum Dei)
- **Typing Defense Games**: Horror-themed typing challenges where users defend against supernatural threats
- **The Crypt**: Document storage system with gothic theming
- **Progress Tracking**: Game-like progression with unlockable levels and rewards
